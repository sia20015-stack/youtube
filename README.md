## 라이브러리 설치

- react 설치 명령어 > 'npx create-react-app (타이틀)'
- react-router-dom 설치 명령어 > 'npm install react-router-dom'
- axios 설치 명령어 > 'npm install axios'
- react-icons 설치 명렁어 > 'npm install react-icons'
- react-player 설치 명령어 > 'npm install react-player'
- sass 설치 명령어 > 'npm install sass'
- react-helment-async 설치 명령어 > 'npm install react-helmet-async'
- swiper 설치 명령어 > 'npm install swiper'


## git 명령어

- git add . > 내 프로젝트에서 어떤 파일을 저장할지 선택하는 명령어 
(.은 모든 파일, 특정파일은 특정파일 이름적기)
- git status .> 내 프로젝트에서 어떤 파일이 저장되는지 터미널창을 통해 육안으로 다시 한 번 더 확인하는 명령어
- git commit -m "(메세지)" > 실제로 저장될 파일을 내 컴퓨터에 저장하는 명령어
- git push - u origin main > 깃허브에 올리기 위해 로컬에 저장된 파일을 깃허브에 올리는 명령어
